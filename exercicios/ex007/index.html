Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase tenho um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica).

Nesta frase temos um termo em itálico usando a tag EM (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro parágrafo temos outro texto marcado no final com a mesma tag MARK usando o parâmetro style para mudar a cor de fundo.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo, usando as tags BIG e SMALL.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase usando a tag INS (semântica).

Existe também o sublinhado com a tag U (não semântica).

Texto sobrescrito

Usamos sobrescrito, tag SUP, para inserir algo do tipo x20+3

Texto subscrito

Ou podemos usar a tag SUB para inserir subscrito como H2O

Código

Uso das tags PRE e CODE para inserir código:


    num = int(input('Digite um número inteiro: '))
    if num % 2 == 0:
        print(f'O número digitado {num} é par.')
    else:
        print(f'O número digitado {num} é ímpar.')
    print('Fim do programa')

Citações

Podemos fazer citação dentro de uma frase, sem deslocamento, usando a tag Q:
Segundo Cecília Meireles, Pelos caminhos do mundo nenhum destino se perde. Há os grandes sonhos dos homens e a surda força dos vermes.
Também podemos inserir a citação com deslocamento usando a tag BLOCKQUOTE:

Pelos caminhos do mundo nenhum destino se perde. Há os grandes sonhos dos homens e a surda força dos vermes.
Também é possível colocar um link dentro da citação para os mecanismos de pesquisa da internet (o link não aparece no texto, mas facilita o SEO achar o livro)
PHP é uma linguagem para a criação de scripts da Web do lado do servidor embutidos na HTML cujo código-fonte é aberto e que é compatível com os mais importantes servidores da Web (mais notadamente o Apache). Abreviações

Usamos a tag ABBR para colocar texto sensitivo em alguma parte do conteúdo. Exemplo: estou estudando HTML para criar sites.
